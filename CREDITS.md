## Open-Source Project Credits

SolvysMIDI builds on free and open-source software. These projects and their contributors made the current product loop possible:

- Basic Pitch by Spotify: audio-to-MIDI transcription.
- Transkun: optional high-accuracy piano transcription mode.
- Audiveris: optical music recognition for PDF score conversion.
- yt-dlp: YouTube media extraction.
- Deno: JavaScript runtime support used by modern yt-dlp extraction.
- FFmpeg: audio conversion and probing.
- Next.js, React, and TypeScript: PWA application framework.
- Vercel Blob: shared MIDI and score storage.
- Tone.js MIDI: MIDI parsing and summary generation.
- OpenSheetMusicDisplay: sheet music rendering support.
- JSZip and fast-xml-parser: MusicXML/MXL parsing support.
- Lucide: interface iconography.
- Playwright: browser verification.
- Fly.io and Vercel: deployment platforms used by the production system.

Dependency credits are acknowledgments, not relicensing. Each upstream project remains governed by its own license.
